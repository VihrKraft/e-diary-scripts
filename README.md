# Скрипты для школьного дневника

Это репозиторий со скриптами для исправления оценок, избавления от замечаний и добавления похвалы в школьном дневнике.

### Как установить

Python c версией 3.12 должен быть уже установлен. 

Также, нужно скачать какую-нибудь базу данных школьного дневника (Например [эту](https://github.com/devmanorg/e-diary/tree/master)).

Существует два способа скачивания: 
1. Скачать проект в виде архива (`Add file` => `Upload files`) и разархивировать его в нужном для вас месте.
2. Сделать `Fork` проекта и клонировать его на компьютер с помощью `GitHub Desktop`.

Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

После так же скачать скрипты и перенести файл `scripts.py` в корневую папку базы данных.

`from datacenter.models import Schoolkid, Subject, Lesson, Mark, Chastisement, Commendation` оставляйте без изменений, если скачилави [эту](https://github.com/devmanorg/e-diary/tree/master) базу данных школьного дневника.

### Описание скриптов

`fix_marks(schoolkid)` - функция, которая исправляет оценки 2 и 3 на 5, принимая на вход Фио ученика (Нужны хотя бы имя и фамилия или имя и отчество):
```
fix_marks('Фролов Иван')
```

`remove_chastisements(schoolkid)` - функция, удаляющая замечания, принимая на вход Фио ученика (Нужны хотя бы имя и фамилия или имя и отчество):
```
remove_chastisements('Фролов Иван')
```

`create_commendation(schoolkid, name_subject)` - функция, создающая похвалу от учителя, принимая на вход Фио ученика (Нужны хотя бы имя и фамилия или имя и отчество) и название предмета (с заглавной буквы):
```
create_commendation('Фролов Иван', 'Математика')
```

### Как запустить

Для запуска базы данных необходимо написать следущее:
```
python manage.py runserver
```

И перейти по ссылке: [http://127.0.0.1:8000/](http://127.0.0.1:8000/).

Чтобы запустить скрипт можно его целиком “копипастнуть” в shell. Для этого нужно создать еще один терминал и активировать `shell`:
```
python manage.py shell
```

А после скопировать весь код из файла `scripts.py` в `shell`. Теперь можно пользоваться скриптами в том же `shell`, например:
```
create_commendation('Фролов Иван', 'Музыка') 
```
Результатом будет похвала, добавленная для Фролова Ивана на сайте по предмету Музыка.
